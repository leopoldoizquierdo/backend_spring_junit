This is an example Backend App, that has been developed as a test for the company intive-FDV.

This app was developed using Java, Spring, Spring Boot, Junit, Jacoco technologies.

## To execute the app and run the unit tests follow the next steps:

### 1) Open the project with Eclipse IDE
### 2) run the command maven install
### 3) Check inside the target folder a new file called my-reports/index.html
### 4) Open it with the browser to check the code coverage with the unit tests
